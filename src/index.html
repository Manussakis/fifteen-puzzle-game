Fifteen Puzzle Game

Rearrange the pieces in order with as fewer moves as possible.

Instructions:

  1. Type how many moves you want the board to be shuffled with (any value between and ).
  2. Click shuffle button and see the pieces shuffling.
  3. Rearrange the board with the same (or even fewer) number of moves that it was shuffled by clicking the pieces that has space to move. The pieces should be in ascending order from left to right, up to down.
